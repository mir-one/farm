Import Farm Settings Data from ZIP

WARNING: MAKE A FARM BACKUP BEFORE ATTEMPTING A SETTINGS IMPORT!

This will allow a ZIP file containing the Farm settings database and custom functions/inputs/outputs/widgets to be uploaded and replace the current settings. Only databases from Farm versions equal or prior to the current Farm version should be imported. There is no check for this, but if you attempt to import a database from a version higher than the current Farm version, there will likely be issues. There may also be issues if the custom controllers are incompatible with the Farm version you're importing in to.

Note 1: This will override all settings, user accounts, and custom functions/inputs/outputs/widgets, and log you out from the web UI. A backup of the current database will be created in the database directory (~/Farm/databases/farm.db.backup_DATE_TIME).

Note 2: During this process, the new database will be upgraded to the current Farm version and all unmet dependencies will be installed. This process can take a while to complete if there are a significant number of dependencies to install, so be patient.
